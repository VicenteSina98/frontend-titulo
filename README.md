# HealthDiagAI

Aplicación web para predecir enfermedades y a qué especialista ir y/o qué medicamentos se pueden suministrar

## Instalación

Requisitos:

```console
npm 9.5.1<= 
```

Instalar dependencias:

```console
npm i
```

## Ejecutar en entorno de desarrollo

```console
npm run dev
```

## Construir entorno de producción

```console
npm run build
```
